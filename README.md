# Semantic Book Recommender

A RAG-powered book recommendation system with a Gradio dashboard. Users describe the kind of book they want, optionally filter by category and emotional tone, and receive AI-generated recommendations with explanations.

## Features

- Semantic search over book descriptions using ChromaDB + OpenAI embeddings
- Emotional tone filtering (Happy, Sad, Angry, Suspenseful, Surprising)
- Category filtering
- Per-recommendation explanations generated by GPT-4o-mini
- Gradio web UI with book cover gallery

## Setup

### 1. Clone & install dependencies

```bash
git clone <your-repo-url>
cd book-recommender
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Configure environment

```bash
cp .env.example .env
# Edit .env and add your OpenAI API key
```

### 3. Add data files

Place the following files in the `data/` directory:

| File | Description |
|------|-------------|
| `books_with_emotions.csv` | Book metadata with columns: `isbn13`, `title`, `authors`, `description`, `thumbnail`, `simple_categories`, `joy`, `surprise`, `anger`, `fear`, `sadness` |
| `tagged_description.txt` | One line per book: `<isbn13> <description text>` |

### 4. Run the app

```bash
python main.py
```

The Gradio dashboard will be available at `http://localhost:7860`.

## Running Tests

```bash
pytest tests/
```

## Architecture

```

    ▼

    ▼

    ▼
GPT-4o-mini generates explanation  (explanation.py)
    │
    ▼
```
